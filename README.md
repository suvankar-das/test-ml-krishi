🌾 KrishiMitra: AI-Powered Smart Agriculture Ecosystem
KrishiMitra is an intelligent farming assistant designed to bridge the gap between traditional farming and modern AI technology. It provides real-time monitoring, disease diagnosis, and data-driven crop recommendations to help small-scale farmers optimize their yields.

🚀 Key Features
Leaf Disease Detection: Upload a photo of a crop leaf and get an instant diagnosis using an optimized TensorFlow Lite (CNN) model.

AI Agricultural Advisor: Powered by Llama 3.3 (via Groq), providing 3-step organic treatment plans and farming advice in multiple languages.

IoT Live Monitoring: Real-time temperature and humidity tracking via ESP32 and DHT22 sensors.

Smart Crop Recommendation: A Machine Learning model (Random Forest) that suggests the best crops based on N-P-K soil values and live climate data.

Digital Ledger: A built-in expense tracker for farmers to manage their financial health.
